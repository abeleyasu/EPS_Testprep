## About This Project

It has 2 user roles now.
- Super admin
- Standard users(students)

It will have mainly:

- E-Learning modules (content and tasks).
- Quizzes and quiz builder.
- Other tools like college resume builder.
- Possibly other features later .


## Project Setup Guide
For Laravel framework:

- composer install
- php artisan migrate
- php artisan serve

For Vue framework:

- npm install
- npm run dev
